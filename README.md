# lpl.ics
LPL 赛程订阅日历

iPhone 用户可以在 设置 -> 密码和账户 -> 添加账户 -> 其他 -> 添加已订阅的日历

Github 订阅源 https://raw.githubusercontent.com/ChengLuffy/lpl.ics/master/LPL2021.ics

Gitee 订阅源 https://gitee.com/chengluffy/lpl2020/raw/master/LPL2021.ics

同步更新，请根据自己的网络环境自行选择。
